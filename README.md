## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS Version Recommended)
- npm or yarn

### Installation

```bash
git clone <your-repository-url>
cd <project-folder>
npm install


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
